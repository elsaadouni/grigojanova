Security Policy

Reporting Security Issues

If you discover a security vulnerability, please report it to us at grigojanova@yas.sh.

We appreciate responsible disclosure and will:

  • Acknowledge receipt of your report within 48 hours
  • Provide an estimated timeline for fixes
  • Notify you when the issue is resolved

Security Measures

  • Content Security Policy (CSP) headers
  • X-Frame-Options to prevent clickjacking
  • HSTS for HTTPS enforcement
  • XSS protection headers
  • Input validation and sanitization
  • Honeypot fields for bot detection
  • Secure external links (noopener noreferrer)

Data Protection

All documents are handled with strict confidentiality. We use secure transmission protocols and do not store sensitive data longer than necessary.
